## Installation

To install and set up the project, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Marina0022/react-test-app.git
    ```
2. Navigate to the project directory:
    ```bash
    cd react-test-app
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```
